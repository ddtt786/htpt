HTPT

HTPT는 html을 ppt처럼 사용하기 위한 목적으로 만들어진 툴입니다.

이 문서는 body 태그에 margin 값이 30px 적용되어 있습니다.
오브젝트와 같은 경우, 화면을 채우는 오브젝트임에도 불구하고 30px정도 공백이 나타날 수 있습니다.

스타일 설정

자동으로 스타일이 적용되는 것들

  • 버튼
  • 글꼴(나눔고딕으로 설정됨)

모든 설정은 class 값으로 이루어집니다.

class=""에서 ""에 지정된 코드를 넣으시면 스타일이 적용됩니다.

코드를 여러개 넣고 싶으시면 class="" 에서 class="abc xyz" 처럼 하시면 됩니다.
예를 들어, dcdo 와 ttl 와 drgx 코드를 적용시키고 싶으시면 class="dcdo ttl drgx"처럼 하세요.

제어 설정

표기 양식: [제어 코드:제어 코드를 넣을 시 미치는 영향]

[drgx:드래그 불가능]

폰트 설정

모든 폰트의 스타일 코드는 4글자로 통일되어 있습니다.

(아래 미리보기 글꼴 표시는 폰트가 잘 보이게끔 볼드 효과를 적용한 바 있습니다.)
표기 양식: [폰트 코드:폰트 코드를 넣을 시 설정되는 폰트]

[btan:바탕]
[dodm:돋움]
[gnsr:궁서]
[gotc:고딕]
[nnpn:나눔손글씨]
[nnsk:나눔스퀘어]
[myzo:나눔명조]
[dcdo:독도]

오브젝트 설정

<div class=""></div> 에서 ""에 지정된 오브젝트 코드를 넣으시면 오브젝트가 생성됩니다. 표기 양식: [오브젝트 코드:오브젝트 코드를 넣을 시 생성되는 오브젝트]

[ttl:홈페이지 환영글](오브젝트 안에 글자를 설정할 수 있음)

[line:단락 나눔]
