HTPT란?

htpt는 html을 ppt처럼 사용하는 것을 목적으로 하는 툴입니다.
htpt는 누구나 쉽게 html으로 웹사이트를 만들기를 바라고 있습니다.

htpt 문법 설명 표기 양식

htpt는 tag와 class값으로 설정이 이루어집니다.
tag 값으로 설정이 이루어지는 것은 [어느 tag를 넣어야 설정이 이루어지는지 : 앞의 tag를 넣으면 어떤 현상이 일어나는지] 로 작성하며,
class 값으로 설정이 이루어지는 것은 {어느 class를 적용해야 설정이 이루어지는지 : 앞의 class를 적용하면 어떤 현상이 일어나는지}로 작성합니다.

보통 오브젝트는 tag로 설정이 이루어지며, 글꼴과 스타일, 제어는 class 값으로 설정이 이루어집니다.

사이트맵과 환영글이 아닌 나머지 태그들은 margin값이 30px 적용되어 있어
화면을 채우는 오브젝트임에도 불구하고 30px정도 공백이 나타날 수 있습니다.

HTPT 문법

제어

  • {drgx : 드래그 불가능}
  • {albr : body tag에 적용하면 html 문서가 어느 길이든지 무조건 스크롤바가 보이게 됨}
  • {clk : a tag가 없어도 커서 모양이 pointer로 변경됨}
  • {sdw : 그림자 효과가 적용됨}

모양

  • {prpl : 프로필 사진처럼 라운드가 쳐짐}
  • {sdw : 그림자가 생김}

스타일

자동으로 스타일이 적용되는 것들

  • 글꼴(구글 노토산스로 설정됨)
  • 단락 나눔(hr tag)
  • 버튼 (button tag)
  • 사이트 로고 (favicon)
  • 사이트 타이틀 (title tag)
  • 사진 (img tag)
  • 선택 상자(select)
  • 인용문 (blockquote tag)
  • 제목, 내용 (h1~h6 tag, p tag)
